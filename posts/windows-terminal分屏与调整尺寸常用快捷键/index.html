Windows Terminal分屏与调整尺寸常用快捷键

2026/4/17 15:54:073 分钟阅读
Windows

Note

此博客灵感来源来自Windows Terminal 分屏与调整常用快捷键总结

此处记录 WindowsTerminal 终端窗口分屏和调整大小的常用操作快捷键。

窗口分屏

将当前的标签页分割为多个独立运行的窗口。

操作项目 默认快捷键 说明
垂直分屏 Alt + Shift + + 将窗口左右平分，增加一个垂直边界
水平分屏 Alt + Shift + - 将窗口上下平分，增加一个水平边界
自动分屏 Alt + Shift + D 自动根据长宽比选额最佳方向进行分屏

调整窗格大小

选中目标窗格后，通过以下组合键微调边界位置。

操作项目 默认快捷键
向左调整大小 Alt + Shift + ←
向右调整大小 Alt + Shift + →
向上调整大小 Alt + Shift + ↑
向下调整大小 Alt + Shift + ↓

导航与管理

在多个分屏之间快速切换或者关闭。

操作项目 默认快捷键 说明
切换焦点 Alt + 方向键 将光标移动到相邻的窗格
关闭窗格 Alt + Shift + W 关闭当前光标所在的窗格

更多设置

使用快捷键CTRL + ,打开设置界面，选择「操作」菜单，修改和查看更多快捷键功能。
