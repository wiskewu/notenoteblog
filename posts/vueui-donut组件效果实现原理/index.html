可以看到，组件是基于SVG进行实现的，大部分数据展示型的图表组件，底层都是基于SVG。SVG优点也很明显，矢量可伸缩，高保真，清晰度高，原生自带DOM事件。

实现原理分析

如果把上面的节点按层级和作用进行划分，可以简化为：

<!--应用主题背景，当前主题是Dark暗黑-->
<div id="donut__3bdd3261" style="background: rgb(26, 26, 26);">
    <div style="position: relative;">
      <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 512 360" style="background: transparent; color: rgb(204, 204, 204);">
        <g class="vue-data-ui-g">
          <!--版权声明-->
          <desc aria-hidden="true">Composed with Vue Data UI 3.18.2</desc>
          <!--定义可复用的渐变-->
          <defs>
            <radialGradient id="gradient_3bdd3261">
              <!--省略stops节点实现-->
            </radialGradient>
          </defs>
          <!--定义可复用的filter-->
          <defs>
            <filter id="blur_3bdd3261" x="-50%" y="-50%" width="200%" height="200%">
              <!--省略filter内部实现-->
            </filter>
          </defs>
          <!--第一层路径-->
          <path stroke="#1A1A1Aff" d="..." fill="#FFFFFF"></path>
          <path stroke="#1A1A1Aff" d="..." fill="#FFFFFF"></path>
          <path stroke="#1A1A1Aff" d="..." fill="#FFFFFF"></path>
          <!--第二层路径-->
          <path class="vue-ui-donut-arc-path" d="..." fill="#1f77b4ff" stroke="#1A1A1Aff" stroke-width="1" filter=""></path>
          <path class="vue-ui-donut-arc-path" d="..." fill="#aec7e8ff" stroke="#1A1A1Aff" stroke-width="1" filter=""></path>
          <path class="vue-ui-donut-arc-path" d="..." fill="#ff7f0eff" stroke="#1A1A1Aff" stroke-width="1" filter=""></path>
          <!--第三层路径-->
          <circle cx="256" cy="180" r="108" fill="url(#gradient_3bdd3261)"></circle>
          <g>
            <!--第四层路径-->
            <path d="..." stroke="#1A1A1Aff" fill="transparent"></path>
            <path d="..." stroke="#1A1A1Aff" fill="transparent"></path>
            <path d="..." stroke="#1A1A1Aff" fill="transparent"></path>
          </g>
        </g>
      </svg>
    </div>

其中，第一层的三个path节点、第二层的三个path节点、第四层的三个path节点，其d路径属性值完全一致，每一层分别使用绘制三个path弧段组成一个圆环。

Note

每层的path数量，与实际数据项数量有关。此处3个path弧段是因为只有三项数据进行展示。

主题

VueUiData 的大部分组件都支持设置Themes，我们这次分析的是Dark主题。主题色配置如下（已简化）：

{
  "dark": {
    "style": {
        "chart": {
            "backgroundColor": "#1A1A1A",
            "color": "#CCCCCC",
            "layout": {},
            "legend": {},
            "title": {},
            "tooltip": {}
        }
    },
    "table": {}
  }
}

可以看到主题色背景应用于以下地方：

  • 背景色，最外层div节点<div id="donut__3bdd3261" style="background: rgb(26, 26, 26);">，即背景色#1A1A1A;
  • 路径的描边，<path stroke="#1A1A1Aff"></path>；
  • 正文的文字颜色，<svg style="background: transparent; color: rgb(204, 204, 204);">，注意svg的背景色为透明以显示主题背景，文字颜色为#CCCCCC;
  • 渐变色stop，<stop stop-color="#1A1A1A00">，注意此处透明度为0;

渲染层级

按照上述的层级划分，可以看到渲染层按照SVG绘制顺序叠加如下：

  1. 第一层：白色底层path
  2. 第二层：彩色（纯色）弧段path
  3. 第三层：应用渐变的circle
  4. 第四层：事件交互层g

第一层

这一层的数据如下：

<!--第一层路径-->
<path stroke="#1A1A1Aff" d="..." fill="#FFFFFF"></path>
<path stroke="#1A1A1Aff" d="..." fill="#FFFFFF"></path>
<path stroke="#1A1A1Aff" d="..." fill="#FFFFFF"></path>

注意到这一层的path填充色均为白色#FFFFFF，与主题无关。这一层的作用为：

  • 颜色混合基底：在第二层中使用了80%的不透明度颜色（最新源码使用100%不透明度纯色填充），使得第二层与第一层进行颜色混合叠加，这样最终颜色效果与背景无关，始终一致。
  • 与filter颜色混合：当鼠标移动到区块上时，其他区块应用filter属性，此时filter的blur模糊导致描边羽化（半透明），羽化的像素混入的是此层的白底，保证被模糊的弧段边界依然清晰可见

第二层

这一层的数据如下：

<!--第二层路径-->
<path class="vue-ui-donut-arc-path" d="..." fill="#1f77b4ff" stroke="#1A1A1Aff" stroke-width="1" filter=""></path>
<path class="vue-ui-donut-arc-path" d="..." fill="#aec7e8ff" stroke="#1A1A1Aff" stroke-width="1" filter=""></path>
<path class="vue-ui-donut-arc-path" d="..." fill="#ff7f0eff" stroke="#1A1A1Aff" stroke-width="1" filter=""></path>

注意到这一层的path填充的是具体每块数据的纯色，并且具备filter属性。假设鼠标移动到第一段path上，效果如下：

此时节点数据如下，增加了filter="url(#blur_3bdd3261)"：

<path class="vue-ui-donut-arc-path" d="..." fill="#1f77b4ff" stroke="#1A1A1Aff" stroke-width="1" filter=""></path>
<path class="vue-ui-donut-arc-path" d="..." fill="#aec7e8ff" stroke="#1A1A1Aff" stroke-width="1" filter="url(#blur_3bdd3261)"></path>
<path class="vue-ui-donut-arc-path" d="..." fill="#ff7f0eff" stroke="#1A1A1Aff" stroke-width="1" filter="url(#blur_3bdd3261)"></path>

即这一层的作用是：

  • 应用各弧段颜色：使用纯色填充
  • 应用filter效果：当鼠标经过时，对其他弧段应用滤镜（模糊+去饱和变灰）

第三层

这一层的数据如下：

<!--第三层路径-->
<circle cx="256" cy="180" r="108" fill="url(#gradient_3bdd3261)"></circle>

这个节点覆盖在所有弧段path之上，用一个共享的径向渐变在环的中间厚度处叠加白色高光，产生3D光泽感。渐变在圆心和外边缘均为透明，只在环厚度中间有白色高光。

第四层

这一层的数据如下：

<g>
  <!--第四层路径-->
  <path d="..." stroke="#1A1A1Aff" fill="transparent"></path>
  <path d="..." stroke="#1A1A1Aff" fill="transparent"></path>
  <path d="..." stroke="#1A1A1Aff" fill="transparent"></path>
</g>

注意到path的填充是fill="transparent"，完全不可见，但位于最顶层。这一层的作用为：

  • 用于捕获所有鼠标事件
  • 当鼠标悬停在该弧段上时，填充色为fill="rgba(0,0,0,0.1)"即#0000001A，产生轻微的深色叠加，给选中弧段一个视觉反馈

单独设立这一层，保证事件捕获的可靠性（第二层的滤镜blur效果可能会影响点击区域），同时与视觉渲染完全解耦。

渐变色

在实现中，渐变色的完整定义如下：

<defs>
  <radialGradient id="gradient_3bdd3261">
    <stop offset="0%" stop-color="#1A1A1A00" stop-opacity="0"></stop>
    <stop offset="58.33333333333333%" stop-color="#FFFFFF00" stop-opacity="0"></stop>
    <stop offset="79.16666666666666%" stop-color="#FFFFFF66"></stop>
    <stop offset="100%" stop-color="#1A1A1A00" stop-opacity="0"></stop>
  </radialGradient>
</defs>

其使用到的地方只有一处，即<circle cx="256" cy="180" r="108" fill="url(#gradient_3bdd3261)"></circle>。 也就是说，整个Donut组件只使用到一个渐变颜色，从元素选择器中可以看到<circle>的大小范围：

按照上述的层级划分，可以看到渲染层按照SVG绘制顺序叠加如下：

  1. 第一层：白色底层path
  2. 第二层：彩色（纯色）弧段path
  3. 第三层：应用渐变的circle

即渐变不是每个弧段单独应用，而是用一个共享的radialGradient圆形覆盖在所有弧段之上。

渐变stop计算逻辑

渐变圆覆盖整个圆环，但只在环的中间厚度处有白色高光，其余区域透明，从而产生3D光泽感。

  • stop 0%: 圆心位置透明，空心内部，透明
  • stop (1 - thickness / radius) * 100%: 圆环的内边缘，透明
  • stop (1 - thickness / radius / 2) * 100%: 圆环的中间，白色高光，具有一定透明度（约80%）
  • stop 100%: 环的外边缘，透明度

弧段路径构造

每个弧段都是一个复合路径：从外弧 -> 连接到内弧起点 -> 内弧（反向）-> 连接回外弧起点，形成环形切片。

AI手搓

让AI根据上面的分析，写了一版简易效果，去除交互层（第四层）和背景主题，可以在本地新建html并复制预览：

<!DOCTYPE html>
<html>

<body>
  <svg id="donut" width="512" height="360" viewBox="0 0 512 360" xmlns="http://www.w3.org/2000/svg">
  </svg>

  <script>
    // ─── 参数 ───────────────────────────────────────────────────────────────────  
    const svgW = 512, svgH = 360;
    const cx = svgW / 2, cy = svgH / 2;

    // 对应 minSize = Math.min(svgW/3, svgH/3)，最小 55  
    const outerR = 120; // ≈ 120  

    // 对应 donutThickness（strokeWidth=128 时 baseRatio=0.25）  
    const thickness = 50;
    const innerR = outerR - thickness;

    const gradientIntensity = 0.4; // gradientIntensity=40 → 40/100  

    // 数据集  
    const dataset = [
      { name: 'A', value: 40, color: '#5C8DCC' },
      { name: 'B', value: 25, color: '#E07B54' },
      { name: 'C', value: 20, color: '#6DBF8A' },
      { name: 'D', value: 15, color: '#C97DC4' },
    ];

    // ─── 工具函数 ────────────────────────────────────────────────────────────────  

    // 极坐标转笛卡尔（从顶部 -90° 开始，顺时针）  
    function polar(r, angleDeg) {
      const rad = (angleDeg - 90) * Math.PI / 180;
      return { x: cx + r * Math.cos(rad), y: cy + r * Math.sin(rad) };
    }

    // 构造环形弧段路径（对应 makeDonut 的 arcSlice）  
    function makeArcSlice(startDeg, endDeg) {
      const os = polar(outerR, startDeg);
      const oe = polar(outerR, endDeg);
      const ie = polar(innerR, endDeg);
      const is_ = polar(innerR, startDeg);
      const large = endDeg - startDeg > 180 ? 1 : 0;
      return [
        `M ${os.x} ${os.y}`,
        `A ${outerR} ${outerR} 0 ${large} 1 ${oe.x} ${oe.y}`,  // 外弧（顺时针）  
        `L ${ie.x} ${ie.y}`,                                     // 连线到内弧终点  
        `A ${innerR} ${innerR} 0 ${large} 0 ${is_.x} ${is_.y}`, // 内弧（逆时针）  
        'Z'
      ].join(' ');
    }

    // hex 颜色追加 80% 不透明度（对应 setOpacity(color, 80)）  
    function setOpacity80(hex) {
      return hex + 'CC'; // CC = 204/255 ≈ 80%  
    }

    // ─── 计算各弧段角度 ──────────────────────────────────────────────────────────  
    const total = dataset.reduce((s, d) => s + d.value, 0);
    const arcs = [];
    let acc = 0;
    for (const d of dataset) {
      const startDeg = acc;
      const endDeg = acc + (d.value / total) * 360;
      arcs.push({ ...d, startDeg, endDeg });
      acc = endDeg;
    }

    // ─── 渐变 stop 偏移量（对应源码公式）────────────────────────────────────────  
    // stop1: (1 - thickness/outerR) * 100  → 环内边缘（透明）  
    // stop2: (1 - thickness/outerR/2) * 100 → 环中间（白色高光）  
    const stop1 = ((1 - thickness / outerR) * 100).toFixed(1) + '%';
    const stop2 = ((1 - thickness / outerR / 2) * 100).toFixed(1) + '%';

    // ─── 构建 SVG ────────────────────────────────────────────────────────────────  
    const svg = document.getElementById('donut');

    // 1. defs：定义径向渐变  
    svg.innerHTML = `  
<defs>  
  <radialGradient id="grad">  
    <!-- 圆心：透明 -->  
    <stop offset="0%"    stop-color="#FFFFFF" stop-opacity="0"/>  
    <!-- 环内边缘：透明（空心边界） -->  
    <stop offset="${stop1}" stop-color="#FFFFFF" stop-opacity="0"/>  
    <!-- 环中间：白色高光 -->  
    <stop offset="${stop2}" stop-color="#FFFFFF" stop-opacity="${gradientIntensity}"/>  
    <!-- 环外边缘：透明 -->  
    <stop offset="100%"  stop-color="#FFFFFF" stop-opacity="0"/>  
  </radialGradient>  
</defs>`;

    // 2. 第一层：白色底（用于颜色混合，对应源码 fill="#FFFFFF"）  
    for (const arc of arcs) {
      const path = document.createElementNS('http://www.w3.org/2000/svg', 'path');
      path.setAttribute('d', makeArcSlice(arc.startDeg, arc.endDeg));
      path.setAttribute('fill', '#FFFFFF');
      path.setAttribute('stroke', '#FFFFFF');
      path.setAttribute('stroke-width', '1');
      svg.appendChild(path);
    }

    // 3. 第二层：彩色弧段，80% 不透明度（对应 setOpacity(arc.color, 80)）  
    for (const arc of arcs) {
      const path = document.createElementNS('http://www.w3.org/2000/svg', 'path');
      path.setAttribute('d', makeArcSlice(arc.startDeg, arc.endDeg));
      path.setAttribute('fill', setOpacity80(arc.color));
      path.setAttribute('stroke', '#FFFFFF');
      path.setAttribute('stroke-width', '1');
      svg.appendChild(path);
    }

    // 4. 第三层：渐变叠加圆（覆盖所有弧段，产生光泽感）  
    // 对应源码：<circle :fill="`url(#gradient_${uid})`" :r="minSize" />  
    const gradCircle = document.createElementNS('http://www.w3.org/2000/svg', 'circle');
    gradCircle.setAttribute('cx', cx);
    gradCircle.setAttribute('cy', cy);
    gradCircle.setAttribute('r', outerR);
    gradCircle.setAttribute('fill', 'url(#grad)');
    svg.appendChild(gradCircle);  
  </script>
</body>

</html>
